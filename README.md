# Pinyin Tool - Pinyin and Translation Assistant

A Chrome extension that lets you select Chinese text on **any webpage** and instantly see pinyin annotations, word-level definitions, and AI-powered translations in a floating overlay. Inspired by reading apps like [Du Chinese](https://www.duchinese.net/), but not locked to curated content. Pinyin Tool works everywhere your browser does: news sites, social media, documentation, emails, and more. It also ships with a built-in EPUB reader and a full-page vocabulary study hub.

---

## Features

### Core

- **Instant pinyin overlay** - Select Chinese text and see pinyin above each word using native HTML ruby annotations.
- **Two-phase rendering** - Local pinyin appears in under 50 ms via [pinyin-pro](https://github.com/zh-lx/pinyin-pro); LLM-enhanced results (definitions + translation) follow a moment later.
- **Clickable word definitions** - Click any word in the overlay to expand a definition card showing its contextual meaning.
- **Full sentence translation** - A natural English translation of the selected passage, generated by the LLM.
- **Polyphonic character disambiguation** - The LLM uses surrounding context to pick the correct reading for characters that have multiple pronunciations.
- **Three pinyin styles** - Tone marks, tone numbers, or no tones.
- **Works offline** - Local-only mode provides pinyin without an API key or internet connection.

### OCR Area Selection

- **Select text from images** - Click "Select text from image" in the popup, drag a rectangle over any region of the page, and the extension runs local OCR ([Tesseract.js](https://github.com/naptha/tesseract.js)) to extract Chinese characters from that region. The recognized text feeds into the same pinyin/translation pipeline.
- **Works on non-selectable content** - Images, canvas elements, screenshots, video thumbnails, embedded PDFs rendered as images, or any other visual content that cannot be highlighted with the mouse.

### Text-to-Speech

- **Listen to pronunciation** - A speaker button on the overlay reads the full selected text aloud with natural pronunciation using the Web Speech API. Reads the entire selection as a sentence, preserving natural speech rhythm and prosody.
- **Configurable** - TTS can be toggled on or off in settings.

### Vocabulary Tracking

- **Save words to your vocab list** - Click the "Add to Vocab" button on any word definition card to save it for later study. Words are stored with their pinyin and contextual definition.
- **Floating vocab card** - A compact floating card shows recently saved words with the option to delete individual entries.

### Library (Reader + Vocab + Flashcards)

- **One unified study page** - Open the Library from the popup to access the EPUB Reader, Vocab list, and Flashcards as three sibling tabs in a single full-page app. Switch between features without losing state.
- **Vocab list** - Browse all saved vocabulary in a spacious layout with large characters, pinyin, definitions, and review stats. Sort by frequency, recency, or alphabetically.
- **Flashcard practice** - Practice saved words with flip-card sessions. Pick a session size, then flip cards to reveal answers. Right/wrong answers are tracked per word.

### Built-in EPUB Reader

- **Read EPUB books with pinyin** - Open EPUB files directly inside the extension and get the same pinyin annotations, word definitions, and translations you get on web pages. No content-script injection issues since the reader runs on the extension's own page.
- **Table of contents and progress tracking** - Navigate chapters via a sidebar TOC. A progress bar tracks your reading position.
- **Reading state persistence** - The reader remembers your position and settings across sessions.
- **Recent books** - Re-open recently read books via the File System Access API without re-selecting the file.
- **Customizable** - Adjustable font size, theme, and content width.

### LLM Providers

- **Multiple LLM providers** - OpenAI, Google Gemini, Ollama (local, no API key needed), or any custom OpenAI-compatible endpoint.

### General

- **Context menu and keyboard shortcut** - Right-click "Show Pinyin and Translation" or press Alt+Shift+P.
- **Theming** - Light, dark, or auto (follows system preference).
- **Configurable font size** - Adjustable from 12 px to 24 px.
- **Response caching** - LLM results are cached for 7 days to save API calls.
- **Style isolation** - The overlay renders inside a Shadow DOM, so it never conflicts with the host page CSS.

## Prerequisites

- **Node.js** v18 or later (with npm)
- **Google Chrome** or any Chromium-based browser (Edge, Brave, Arc, etc.)
- *(Optional)* An API key from [OpenAI](https://platform.openai.com/api-keys) or [Google Gemini](https://aistudio.google.com/app/apikey), **or** a local [Ollama](https://ollama.com/) installation, if you want LLM-enhanced translations and definitions

## Setup

### 1. Clone the repository

```bash
git clone <repository-url>
cd "Du Chinese Plugin"
```

### 2. Install dependencies

```bash
npm install
```

### 3. Build the extension

```bash
npm run build
```

This produces a production-ready `dist/` folder. For development with automatic rebuilds on file changes, use:

```bash
npm run dev
```

### 4. Load the extension in Chrome

1. Open `chrome://extensions/` in Chrome.
2. Enable **Developer mode** (toggle in the top-right corner).
3. Click **Load unpacked**.
4. Select the `dist/` folder inside the project directory.
5. The Pinyin Tool icon should appear in your browser toolbar.

> **Tip:** After rebuilding, return to `chrome://extensions/` and click the refresh icon on the Pinyin Tool card to pick up changes.

## Configuration

Click the Pinyin Tool icon in the toolbar to open the settings popup. All settings are synced across Chrome instances via `chrome.storage.sync`.

### LLM Provider

Choose which AI service powers translations and definitions. Selecting a provider auto-fills the base URL and default model.

| Provider | API Key Required | Local / Cloud | Default Model | Base URL |
| -------- | ---------------- | ------------- | ------------------- | ------------------------------------------- |
| OpenAI | Yes | Cloud | gpt-4o-mini | https://api.openai.com/v1 |
| Gemini | Yes | Cloud | gemini-2.0-flash | https://generativelanguage.googleapis.com |
| Ollama | No | Local | qwen2.5:7b | http://localhost:11434/v1 |
| Custom | Optional | Varies | User-specified | User-specified |

**Ollama** is free and runs entirely on your machine with no API key and no cloud dependency. Install it from [ollama.com](https://ollama.com/), pull a model (e.g. `ollama pull qwen2.5:7b`), and you are ready to go. The extension automatically detects your installed Ollama models and populates the Model dropdown -- no need to remember model names.

**Custom** accepts any OpenAI-compatible endpoint (Azure OpenAI, Together AI, Groq, a self-hosted vLLM server, etc.). Set the base URL and model manually.

### Other Settings

| Setting | Options | Default |
| ---------------- | ---------------------------------------------------- | ------------ |
| API Key | Your provider API key (hidden for Ollama) | none |
| Base URL | Auto-filled from provider; can be overridden | Per provider |
| Model | Dropdown per provider (Ollama auto-fetches installed models); refresh button + custom text input | Per provider |
| Pinyin Style | Tone marks / Tone numbers / No tones | Tone marks |
| Font Size | 12 px to 24 px slider | 16 px |
| Theme | Light / Dark / Auto | Auto |
| Enable LLM Mode | On / Off | On |
| Enable TTS | On / Off | On |

When **LLM Mode** is off, the extension only shows local pinyin (no definitions or translations) and requires no API key or internet.

## Usage

### Popup Actions

The popup provides three top-level actions:

1. **Settings** - Configure your LLM provider, pinyin style, theme, and other options.
2. **Select text from image** - Activates OCR area selection mode. Drag a rectangle over any part of the page to extract text from images or non-selectable content.
3. **Open Library** - Opens the full-page Library in a new tab, with sibling tabs for the EPUB Reader, the Vocab list, and Flashcards.

### Triggering the Pinyin Overlay

There are three ways to activate the overlay on any webpage:

1. **Select text** - Highlight Chinese text on any webpage with your mouse or keyboard; the overlay appears automatically near the selection.
2. **Right-click** - Select text, then right-click and choose **Show Pinyin and Translation** from the context menu.
3. **Keyboard shortcut** - Select text, then press `Alt+Shift+P`.

### Reading the Overlay

The overlay appears in two phases:

1. **Phase 1 (instant)** - Pinyin annotations appear above each word immediately. If LLM mode is on, a loading indicator shows where the translation will appear.
2. **Phase 2 (approximately 1-3 seconds)** - The LLM result arrives with contextually accurate pinyin, per-word definitions, and a full English translation. The overlay updates in place.

If LLM mode is off or no API key is configured, only Phase 1 runs.

### Interacting with the Overlay

| Action | Result |
| --------------------------- | ------------------------------------------------- |
| Click a word in the overlay | Expands a definition card with contextual meaning |
| Click "Add to Vocab" | Saves the word to your vocabulary list |
| Click the speaker button | Reads the full selected text aloud (TTS) |
| Click outside the overlay | Dismisses the overlay |
| Press Escape | Dismisses the overlay |
| Select new text | Replaces the current overlay with a new one |

### Using the EPUB Reader

1. Open the Library from the popup. The Reader tab is shown by default.
2. Select an `.epub` file from your device (drag-and-drop or file picker).
3. Read the book. Select any Chinese text within the reader to trigger the same pinyin overlay used on web pages.
4. Navigate chapters via the table of contents sidebar. Your reading position is saved automatically.
5. Recently opened books appear for quick re-opening without re-selecting the file.
6. Switch to the Vocab or Flashcards tab at any time without losing your reading position.

### Notes

- Selections containing no Chinese characters are silently ignored (no overlay appears).
- Mixed text (Chinese + English/numbers/punctuation) is handled gracefully: non-Chinese segments are preserved as-is.
- Selections longer than 500 characters are truncated for the LLM request; local pinyin still processes the full text.

## Packaging for Distribution

```bash
npm run build
cd dist
zip -r ../pinyin-tool-extension.zip .
```

To publish on the Chrome Web Store:

1. Go to the [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole).
2. Register a developer account (one-time $5 fee).
3. Upload `pinyin-tool-extension.zip`.
4. Fill in the store listing (description, screenshots, privacy disclosures).
5. Submit for review (typically 1-3 business days).
